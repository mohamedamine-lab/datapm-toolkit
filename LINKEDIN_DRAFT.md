# LinkedIn Launch Post — DataPM Toolkit

---

Data PMs, soyez honnêtes : combien de temps passez-vous à écrire des Project Charters, des frameworks KPI ou des Data Specs... à partir de zéro ?

Moi, j'en avais marre.

Alors cette nuit, j'ai construit **DataPM Toolkit** — un outil IA qui génère des artefacts de projet data en quelques secondes.

🎯 Vous décrivez votre projet
📋 Vous choisissez le type de livrable  
✨ L'IA génère un document structuré, actionnable, prêt pour vos stakeholders

Disponible maintenant en beta gratuite :

👉 https://datapm-toolkit.vercel.app

4 artefacts disponibles :
• Project Charter (avec RACI, risques, timeline)
• KPI Framework (formules, targets, data sources)
• Data Specification (modèle de données, règles qualité, lineage)
• Stakeholder Deck Outline (slide par slide avec talking points)

Pas de compte. Pas de CB. Juste votre contexte projet → un livrable pro.

C'est un v0.1, c'est gratuit, et j'aimerais vos retours.

Testez, cassez, dites-moi ce qui manque. 🙏
